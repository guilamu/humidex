# Comparateur d'humidex

Page web statique pour comparer l'humidex interieur et exterieur.

## Utilisation

1. Ouvrir `index.html` dans un navigateur.
2. Saisir la temperature et l'humidite interieure.
3. Saisir la temperature et l'humidite exterieure.
4. Lire la recommandation d'aeration.

## Publication

Ce projet peut etre heberge directement sur GitHub Pages sans build ni dependances.
